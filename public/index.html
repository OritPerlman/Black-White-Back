Clothes API Documentation

Users Routes

Method URL Body Description
GET /users
GET /users/usersList Get a List of all users that only admin can access
GET /users/:userId Get data of specific user only admin can access
GET /users/myInfo Get the details of user by his token
POST /users 
{
"firstName":"test",
"lastName":"test",
"email":"test@test.com",
"password":"123456"
}
Add a new user to the data base
POST (login) /users/login 
{
"email":"test@test.com",
"password":"123456"
}
User login, if the email and password are correct the server will respond with a token
DELETE /users/:userId Delete user only admin can delete
PUT /users/:userId update user
PATCH /users/changeRole/:userId changing the role of the user
PATCH /users/changeActive/:userId changing the active of the user
